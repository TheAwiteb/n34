# Convert to Draft

> `n34 pr draft` command

**Usage:**
```
Usage: n34 pr draft [OPTIONS] <PR_ID>

Arguments:
  <PR_ID>  The open PR id to draft it

Options:
      --repo <NADDR-NIP05-OR-SET>  Repository addresses
```

Issue a kind `1633` (Draft status) for the specified PR. The PR have to
be open.
